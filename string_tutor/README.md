### Конструкторы и базовые операции



| Метод / операция         | Назначение                           | Пример                     |
| ------------------------ | ------------------------------------ | -------------------------- |
| `std::string s;`         | Пустая строка                        | —                          |
| `std::string s(n, c);`   | Строка из `n` повторений символа `c` | `string(5, '*') → "*****"` |
| `s.size()`, `s.length()` | Длина строки                         | `s.size()`                 |
| `s.empty()`              | Проверка на пустоту                  | `if (s.empty()) …`         |
| `s.clear()`              | Очистка строки                       | `s.clear()`                |
| `s += t`, `s + t`        | Конкатенация                         | `"Hello" + "!"`            |

### Доступ к символам



| Метод                   | Назначение                                       | Пример       |
| ----------------------- | ------------------------------------------------ | ------------ |
| `s[i]`                  | Доступ по индексу (без проверки границ)          | `s[0] = 'A'` |
| `s.at(i)`               | Доступ с проверкой границ (`std::out_of_range`)  | `s.at(0)`    |
| `s.front()`, `s.back()` | Первый / последний символ (если строка не пуста) | `s.front()`  |

### Изменение содержимого



| Метод                    | Назначение                   | Пример                  |
| ------------------------ | ---------------------------- | ----------------------- |
| `s.push_back(c)`         | Добавить символ в конец      | `s.push_back('!')`      |
| `s.pop_back()`           | Удалить последний символ     | `s.pop_back()`          |
| `s.append(t)`            | Добавить строку/часть строки | `s.append("abc")`       |
| `s.insert(pos, t)`       | Вставка подстроки по позиции | `s.insert(2, "XY")`     |
| `s.erase(pos, len)`      | Удаление части строки        | `s.erase(1, 3)`         |
| `s.replace(pos, len, t)` | Замена фрагмента             | `s.replace(0, 5, "Hi")` |
| `s.substr(pos, len)`     | Извлечение подстроки         | `s.substr(1, 3)`        |

### Поиск



| Метод                        | Назначение                                                   |
| ---------------------------- | ------------------------------------------------------------ |
| `s.find(t)`                  | Поиск подстроки/символа (возвращает `size_t`, `npos` при неудаче) |
| `s.rfind(t)`                 | Поиск с конца                                                |
| `s.find_first_of(chars)`     | Поиск первого символа из набора                              |
| `s.find_first_not_of(chars)` | Поиск первого символа **не** из набора                       |

### Работа с отдельными символами (`<cctype>`)



| Функция                              | Назначение                                      | Пример                |
| ------------------------------------ | ----------------------------------------------- | --------------------- |
| `std::isalpha(c)`                    | Буква латинского алфавита                       | `isalpha('A') → true` |
| `std::isdigit(c)`                    | Цифра                                           | `isdigit('5') → true` |
| `std::isalnum(c)`                    | Буква или цифра                                 | `isalnum('7') → true` |
| `std::islower(c)`, `std::isupper(c)` | Строчная / заглавная буква                      | —                     |
| `std::tolower(c)`, `std::toupper(c)` | Преобразование регистра                         | `toupper('a') → 'A'`  |
| `std::isspace(c)`                    | Пробельный символ (`' '`, `'\t'`, `'\n'` и др.) | —                     |

> ⚠️ Все функции принимают `int`, но безопасно передавать `char` (после приведения к `unsigned char` при работе с не-ASCII).

### Алгоритмы для строк (`<algorithm>`)



| Алгоритм                            | Назначение                   | Пример                  |
| ----------------------------------- | ---------------------------- | ----------------------- |
| `std::sort(s.begin(), s.end())`     | Сортировка символов в строке | Для ключа анаграмм      |
| `std::reverse(s.begin(), s.end())`  | Разворот строки              | Для проверки палиндрома |
| `std::count(s.begin(), s.end(), c)` | Подсчёт вхождений символа    | —                       |
| `std::find()` / `std::find_if()`    | Поиск по условию             | —                       |

### Ввод-вывод строк (`<iostream>`)



| Операция                    | Назначение                      | Особенности                        |
| --------------------------- | ------------------------------- | ---------------------------------- |
| `std::cin >> s`             | Чтение слова до пробела         | Пропускает начальные пробелы       |
| `std::getline(std::cin, s)` | Чтение строки **до `\n`**       | Сохраняет пробелы внутри           |
| `std::cin.get()`            | Чтение одного символа           | Полезно для посимвольной обработки |
| Чтение до EOF               | `while (getline(cin, s)) { … }` | Для многострочного ввода           |
